# SimplePad v1.0.0

**Editor de texto multiplataforma minimalista com abas.**

Primeiro release estável — baixe os instaladores abaixo e comece a anotar.

## O que vem nesta versão

### Editor

- Abas com drag & drop, indicador de alterações e atalhos familiares
- Monaco Editor (undo/redo por aba, Markdown e texto puro)
- Sessão restaurada ao reabrir o app

### Markdown

- Split View Editor | Preview (GFM: tabelas, checklists, código…)
- Exportar como **HTML** ou **PDF**

### Produtividade

- Configurações (fonte, tema, auto-save)
- Arquivos recentes
- Modo Distração Zero (F11)
- Auto-update via GitHub Releases

### Plataformas

- **Windows** — instalador NSIS + portable
- **macOS** — DMG + ZIP
- **Linux** — AppImage + deb

## Instalação rápida

1. Baixe o arquivo da sua plataforma nos _Assets_ desta release
2. Instale / execute
3. (macOS) Se o Gatekeeper bloquear: _Ajustes → Privacidade e Segurança → Abrir mesmo assim_ (build sem notarização por enquanto)

## Limitação conhecidas

- Tamanho do instalador > 70 MB (Electron + Monaco)
- Notarização Apple ainda não habilitada por padrão
